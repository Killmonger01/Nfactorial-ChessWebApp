# ♟ Chess — Премиальная шахматная платформа

> Современное веб-приложение для игры в шахматы с ИИ-тренером, мультиплеером и монетизацией.

🔗 **Живой сайт:** [nfactorial-chess-web-app-q5zk.vercel.app](https://nfactorial-chess-web-app-q5zk.vercel.app)  
📦 **Репозиторий:** [github.com/Killmonger01/Nfactorial-ChessWebApp](https://github.com/Killmonger01/Nfactorial-ChessWebApp)


## Что реализовано

### 🎮 Игровой движок
- Полные правила шахмат: рокировка, взятие на проходе, мат, пат, шах
- Игра вдвоём на одном экране (PvP)
- Игра против ИИ на базе **Stockfish 18 (WASM)** — три уровня сложности: Easy / Medium / Hard
- Выбор цвета фигур перед игрой против ИИ (белые / чёрные / случайно)
- История ходов в алгебраической нотации

### 🤖 AI Coach
После завершения партии пользователь может найти партию в истории игр и при и каждый ход ai будет анализировать.
это функция открывается после оплаты. Для теста можно оплатить тестовой картой
номер карты: 4242 4242 4242 4242, дата: 12/34, CVC: 123

### 🔗 Мультиплеер по ссылке
- Создаёшь партию — получаешь уникальную ссылку
- Друг открывает ссылку (с телефона или компьютера) — игра начинается мгновенно
- Реализовано через **Supabase Realtime** (без WebSocket-сервера)
- Доска автоматически переворачивается для игрока чёрными
- Работает без регистрации

### 👤 Авторизация и профиль
- Регистрация / вход через **Supabase Auth** (email + пароль)
- Профиль с личной статистикой: игры, победы, поражения, ничьи, процент побед
- История всех сыгранных партий с возможностью просмотра записи каждой
- Данные хранятся в **PostgreSQL (Supabase)** с защитой через Row Level Security

### 💳 Монетизация (Stripe)
- Тарифный план **Pro** за $4.99/мес
- Оплата через **Stripe Checkout** (тестовый режим)
- После оплаты статус `is_pro` записывается в БД через webhook
- В шапке сайта: кнопка «Upgrade ✨» для обычных пользователей, золотой бейдж «✨ Pro» для подписчиков
