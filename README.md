## Что это и для кого

**Chess** — это готовый прототип стартапа, а не просто учебная доска. Продукт нацелен на трёх пользователей:

- **Новичок** — хочет научиться играть, получить разбор своих ошибок
- **Любитель** — хочет сыграть с другом по ссылке прямо сейчас, без регистрации
- **Энтузиаст** — хочет тренироваться против сильного ИИ и следить за своей статистикой

---

## Что реализовано

### 🎮 Игровой движок
- Полные правила шахмат: рокировка, взятие на проходе, мат, пат, шах
- Игра вдвоём на одном экране (PvP)
- Игра против ИИ на базе **Stockfish 18 (WASM)** — три уровня сложности: Easy / Medium / Hard
- Выбор цвета фигур перед игрой против ИИ (белые / чёрные / случайно)
- История ходов в алгебраической нотации

### 🤖 AI Coach — уникальная фича
После завершения партии пользователь может нажать **«Analyze with AI»** прямо в режиме просмотра. **Google Gemini 2.5 Flash** анализирует всю партию и даёт текстовый разбор: где была допущена ошибка, какой ход был бы лучше, общая оценка игры. Это то, чего нет на большинстве шахматных сайтов в свободном доступе.

### 🔗 Мультиплеер по ссылке
- Создаёшь партию — получаешь уникальную ссылку
- Друг открывает ссылку (с телефона или компьютера) — игра начинается мгновенно
- Реализовано через **Supabase Realtime** (без WebSocket-сервера)
- Доска автоматически переворачивается для игрока чёрными
- Работает без регистрации

### 👤 Авторизация и профиль
- Регистрация / вход через **Supabase Auth** (email + пароль)
- Профиль с личной статистикой: игры, победы, поражения, ничьи, процент побед
- История всех сыгранных партий с возможностью просмотра записи каждой
- Данные хранятся в **PostgreSQL (Supabase)** с защитой через Row Level Security

### 💳 Монетизация (Stripe)
- Тарифный план **Pro** за $4.99/мес
- Оплата через **Stripe Checkout** (тестовый режим)
- После оплаты статус `is_pro` записывается в БД через webhook
- В шапке сайта: кнопка «Upgrade ✨» для обычных пользователей, золотой бейдж «✨ Pro» для подписчиков

### 🎨 Дизайн «Obsidian Jade»
Авторская тёмная тема, вдохновлённая Chess.com и Lichess, но с уникальной палитрой:
- Фон: глубокий чёрный `#070b0f` с изумрудными радиальными свечениями и точечной текстурой
- Акцент: изумрудный `#10b981` + золото `#f0a500`
- Доска: рамка «грецкий орех» с многослойными тенями
- Glassmorphism на сайдбаре, модалках и выпадающих меню
- Адаптивный дизайн — удобно играть с телефона
- Корректное отображение шахматных фигур на iOS/Android (решена проблема emoji-рендеринга)

## Почему это ценно

Большинство шахматных сайтов либо слишком сложные (Chess.com, Lichess), либо слишком простые (учебные проекты). Этот продукт занимает нишу **«быстро поиграть с другом + получить разбор от ИИ»** — без лишней регистрации, без громоздкого интерфейса, с красивым дизайном, который приятно открыть с телефона.

Потенциальные направления монетизации и роста:
- Подписка Pro (уже реализована)
- Кастомные скины для фигур и досок
- Турниры с призовым фондом во внутренней валюте
- «Шахматы для подготовки к алгоритмическим интервью» — уникальная ниша
